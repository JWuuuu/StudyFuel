StudyFuel
Prototype UI for an AI powered study assistant concept. Current repo contains the front end mockup and feature flow walkthrough. Backend and generation logic are planned but not implemented yet.
Current features
Responsive landing page and feature flow sections
UI mockups for note to summary, quiz generation, and progress tracking
Planned
API backend for note ingestion and content generation
User accounts and progress persistence
