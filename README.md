# StudyFuel

**Live demo:** https://jwuuuu.github.io/StudyFuel/

UI concept page for an AI-powered study assistant. This repo is a front-end prototype that explains the intended user flow (notes → summary → quiz → progress). No backend, data persistence, or AI generation is implemented yet.

## What exists today
- Responsive landing page (mobile + desktop)
- Walkthrough sections describing the planned product flow and screens

## Not implemented yet
- AI summarization / quiz generation
- Accounts, storage, and progress tracking
- Backend APIs

## Tech stack
- HTML
- CSS
- JavaScript
- GitHub Pages
